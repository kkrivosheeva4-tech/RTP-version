Настройка двухфакторной аутентификации

Для настройки 2FA используйте приложение-аутентификатор (Google Authenticator, Authy и т.п.). Отсканируйте QR-код или введите секретный ключ вручную.

QR-код появится после подключения API
Секретный ключ для ручного ввода в приложении аутентификатора

Инструкция

  1. Установите приложение-аутентификатор на телефон.
  2. Отсканируйте QR-код выше или введите секретный ключ вручную.
  3. Введите 6-значный код из приложения для подтверждения.
  4. Нажмите «Завершить настройку».
